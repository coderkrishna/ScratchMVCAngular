Calculator Application

The following requirements are to be taken into consideration when this application was being built:

  • The numbers are to be entered with commas in between them
  • Must display the metrics that are calculated after the numbers are entered

The metrics that are going to be calculated:

  • Mean: The mean is the average of the numbers that you enter in.
  • Median: The median is known as the middle of the list, and the list should be sorted ahead of time!
    • If the list has an odd length of integers, then the median is the exact middle number in the list
    • However, if the list has even length of integers, then the average of the middle two numbers results in the median of the list
  • Mode: This means the number or numbers that occur the most in the list that you as the user puts in. There may be a couple of scenarios with regards to that.
    1. No mode or mode value of 0 - This means that there is no mode from the list that is entered
    2. A single mode value that is greater than 0 - This means that an integer x has been repeated more than once inside of the list
    3. Multiple mode values that are greater than 0 - This means that there exists a set of integers which is a subset of the list where each integer in the list occurs more than once
